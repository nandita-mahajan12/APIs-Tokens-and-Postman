APIs, Tokens, and Postman
Tasks
- Task 1: Set up Postman and make a GET request to a public API such as JSONPlaceholder.
- Task 2: Use Postman to send a POST request with JSON data to a local Flask application.
- Task 3: Build Flask APIs supporting GET, POST, PUT, and DELETE methods and test them using Postman.
- Task 4: Implement Bearer Token authentication in a Flask API.
- Task 5: Fetch user information from a public API such as the GitHub user profile endpoint.
- Task 6: Create a Flask API that accepts a number via POST request and returns its square.
